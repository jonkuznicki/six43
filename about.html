About

SIX43 is built for youth baseball decisions.

Not more “tips.” Not more yelling. Just a clear way to teach players (8–14) what to do before the ball is hit — so innings don’t snowball.

Start the Learn Path Browse Articles Coaching Tools

What we focus on

Most youth games aren’t decided by exit velo. They’re decided by: hesitation, wrong throws, missed covers, and extra bases.

SIX43 is a simple framework for teaching Baseball IQ — the small decisions that create calm outs and stop chaos innings.

  • Pre-pitch thinking (who has what? where’s the play?)
  • Sure outs (take the out, don’t force the hero play)
  • Communication (short words, same meaning, every time)
  • Game-speed reps (habits that hold up on Saturdays)

Who this is for

Coaches and parents working with players from 8U to 14U — especially teams that want to play cleaner defense and stop giving away runs.

How to use the site

  • Start with the Learn Path
  • Pick one idea for the week
  • Teach it with one cue + one drill
  • Measure it in games: fewer freebies, faster outs
