The SIX43 90-minute practice template

0–10 min — Dynamic warmup Movement + footwork + sprint mechanics. Simple cues. Fast pace.
10–25 min — Throwing progression Partner throws → quick feet → long toss (if space). Protect arms first.
25–55 min — Defensive reps High-rep infield/outfield blocks. Short lines. Clear jobs.
55–75 min — Team defense / situations Cutoffs, relays, bunt D, rundowns. Build calm decisions.
75–90 min — Fun finisher Competitive game that reinforces today’s focus (and ends on energy).
11–12U default (scales 8–14) Same structure, adjust language + distance + speed based on age.
